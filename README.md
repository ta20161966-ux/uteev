# Currency Converter

## Автор
Иванов Иван

## Описание
Приложение для конвертации валют с GUI, использованием внешнего API, сохранением истории операций в JSON и интеграцией с Git.

## Требования
- Python 3.8+
- Библиотеки: tkinter, requests

## Установка и запуск
1. Установите зависимости:
   pip install -r requirements.txt
2. Получите API-ключ на exchangerate-api.com и вставьте его в config.py или main.py.
3. Запустите приложение:
   python main.py

## Использование
1. Выберите валюты из выпадающих списков.
2. Введите сумму (только положительное число).
3. Нажмите «Конвертировать».
4. История операций отображается в таблице и сохраняется в history.json.

## Примеры использования
- Конвертация 100 USD в EUR.
- Проверка истории после нескольких операций.
